À propos
CosmOS est un projet complétement Open-source et gratuit vous permettant de stocker et de gérer vos fichiers en ligne. Afin de mieux définir la plateforme, CosmOS est bâti comme un WebOS, c'est-à-dire une plateforme reproduisant le fonctionnement général d'un OS classique comme Windows ou Linux mais en ligne. À l'aide de son architecture, CosmOS vous permet d'accéder à une multitude d'applications vous permettant de gérer mais aussi d'éditer, lire, visionner et même vous détendre.
Le projet n'en est encore qu'à ses balbutiements mais j'espère réussir à vous fournir une plateforme la plus simple, rapide et fonctionnelle possible. Vous pouvez, si vous le souhaitez, suivre le développement de la plateforme via GitHub ou faire un don si vous voulez contribuer au développement du projet.
Pour information, la plateforme est entièrement développée et testée pour Firefox 56 et +. Si vous repérez un bug sur un autre navigateur ou une ancienne version de Firefox, vous pouvez me contactez à l'adresse suivante : romain.claveau@protonmail.com.

A bientôt sur CosmOS !
